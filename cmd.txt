How to install Python and Jupyeter on Vscode in Ubuntu 22.04 ?
FOR PYTHON : 
sudo apt install python3
python3 --version

FOR VSCODE : 
GO To BROWSER AND PASTE  
https://code.visualstudio.com/docs/?dv=linux64_deb

cd Downloads/
sudo apt install ./code_1.84.1-1699275408_amd64.deb

FOR KERNEL :
sudo apt-get -y install python3-ipykernel

FOR NUMPY and PANDAS  using APT package: 
sudo apt install python3-pandas
sudo apt install python3-numpy

FOR MATPLOTLIB : 
sudo apt-get install python3-matplotlib
sudo apt-get install python3-sklearn
